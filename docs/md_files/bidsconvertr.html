BIDSconvertR

The hexagonal sticker is created with the iconspackage and based on the MRI svg graphics provided by Flaticon and was created by mavadee FlaticonLink.

Aim

The goal of BIDSconvertR is to provide a workflow, which is able to:

  • convert DICOM data to NIfTI data using dcm2niix

  • structure this data according to the BIDS specification

    • validate the sequence-ID’s

    • enable easy access to the BIDS-Validator (Website/Docker)

  • provide the papayaWidget viewer for inspecting the images

  • enable continuous application during data acquisition in ongoing studies

Features

Renaming of unclean subject-ID’s or session-ID’s.

Everytime new files or sequences are added, the ‘sequence mapper’ opens again until everything is declared according to BIDS. Already processed files are skipped.
